汽车工程学院2015年学风建设工作总结

2015年，是学校优良学风重点建设第三年，汽车工程学院学生工作瞄准学生数量规模逐年稳定的新常态，立足人才培养质量中心，注重队伍建设，强化组织保障，创新教育管理模式，深化目标管理，实施细化管理，巩固学风建设成果。学院学生管理教育工作不断开创新格局。

2015年，在学校第二届百名标兵评选中，学院共有9人获奖，位列全校十二个学院第四。在学校田径运动会中，学院再次获男团第一、团体总分第二的好成绩。在2014-2015学年度本科生综合测评工作中，获优秀学生标兵2人，占全校18.2%；其他先进个人404人，获奖比例达34.97%。学院涌现出1名优秀学生党员、29名优秀共青团干部和177名优秀共青团员。学院2015届毕业生最终就业率99.85%，4人成功创业，学院最终获得学校就业先进单位一等奖，并蝉联这一奖项五连冠。此外，学院获得第五届宿舍文化节“优秀组织奖”， 2013-2014年度心理健康教育工作先进单位、第六届5.25心理健康教育月系列活动最佳承办奖。在2015级新生军训中，获得分列式第一名和第三名，两个先进集体、14间优秀内务宿舍和55名优秀学员得到表彰。

2015年我院学生参与各类校外学术科技、文化艺术赛事达18项，获奖学生达67人次,较上年增加28.8%。在全国三维化创新设计大赛中获得国家级一等奖，FSAE-华汽车队在2015中国大学生方程式汽车大赛中，在全国参赛70多所高校中获得总成绩三等奖和营销报告第三名的好成绩。在第九届广东省大中专学生科技学术节“南网杯”第八届广东省大学生节能减排社会实践与科技大赛中取得了一等奖的好成绩。

 

一、 学院基本情况

截至2015年12月31日，学院三个专业四个年级共有在校生1420人，现有党总支1个，下设教工党支部4个，学生党支部2个；团委、学生会各1个，行政班级32个。辅导员6人，就业专员1人，班主任29人。2015年毕业学生665人，入学新生278人。

 

二、2015年学风建设工作主要做法

（一）构建长效机制，打造高质量人才培养目标

1、查漏补缺，全面完善学风管理制度

2015年汽车工程学院在总结经验、深入分析的基础上，立足一个方案，完善四项制度，实施四项目标管理，扭住三个重点关注，固化六个机制，落实人员三级责任制，“三个阵地”常抓不懈，继续坚持深入开展学风建设工作。针对学风建设中疏漏的方面进行补充完善，加强学生周末及节假日管理，特殊学生重点关注，完善和修订《汽车工程学院学风建设实施方案》、三个阵地建设相关三个文件《汽车工程学院关于进一步整顿规范课堂秩序的管理办法》、《汽车工程学院关于创建学习型宿舍的管理规定》、《汽车工程学院关于进一步严格规范考试工作的规定》。补充修订了《汽车工程学院周末及节假日学生管理规定》、《汽车工程学院高年级住校及回校学生管理规定》、《汽车工程学院高年级学生行为承诺书》《助学金获得者感恩承诺书》和《华南理工大学广州学院汽车工程学院优良学风先进班集体、学习型宿舍、优良学风先进个人评选办法》。

通过层级制定目标，落实完善制度管理，学生在课堂、宿舍、考场，平时在校及周末节假日均能做到做到主动遵守学院规章制度，严格规范自身行为。全院倡导无手机课堂、学期中后期晚自习、成立帮扶对子等措施，低年级班级针对高数、机械制图、大学物理等重点难点科目组织平时辅导、阶段性复习小考等措施帮助学生打好基础。在2015年上半年进行的学风总结表彰中，有1个班级获得优良学风先进班集体标兵，6个班级获得优良学风先进班集体；还表彰学习型宿舍标兵16间、学习型宿舍55间；优良学风先进个人标兵51名、优良学风先进个人115名。

2、鼓励学术科技类活动，营造良好育人氛围

为营造良好的学术科技氛围，学院鼓励教师、学生参与各类学术科技讲座、竞赛等活动。学院为华汽车队、飞思卡尔、节能减排、物流大赛等项目团队提供经费、场地保障，指派专门的指导老师，鼓励学生积极参与项目团队。学院举办专业知识讲座7场次，举行为期一个月的新生辩论赛，承办了“郑明杯”第五届全国大学生物流设计大赛校内选拔赛，推动了学术科技工作发展。

为营造良好的学术科技氛围，学院鼓励教师、学生参与各类学术科技讲座、竞赛等活动。学院为华汽车队、飞思卡尔、节能减排、物流大赛等项目团队提供经费、场地保障，指派专门的指导老师，鼓励学生积极参与项目团队。学院举办专业知识讲座7场次，举行为期一个月的新生辩论赛，承办了“郑明杯”第五届全国大学生物流设计大赛校内选拔赛，推动了学术科技工作发展。

3、宣传先进学优秀，榜样带动力量大

学院加大对党、团及各类校内外获奖的优秀教师和学生的宣传引导，促进全员共同进步。学院通过召开表彰会、建立荣誉册、更新荣誉墙、充实宣传栏、及时分享经验等方式，宣传和引导广大学生学先进、赶先进，倡导正气、传播正能量。各党支部定期召开支部民主生活会，开展批评与自我批评。通过宣传和学习先进团组织，先进班集体标兵、优良学风先进班集体、学习型宿舍、优秀党员、团员、优秀学生标兵、奖学金获得者等先进集体和个人的过程，倡导正气、传播正能量，形成良好的创先争优氛围。据统计，教工党员和学生党员中，获得各类校内外奖励和“三好学生”、国家企业奖学金的比例分别达到95%和98%。

 

（二）注重队伍建设，辅导员、班主任队伍形成合力

1、针对特点开展工作，学生管理精细化

以队伍建设为基础，注重在工作中加强培训、交流和总结，学生工作队伍素质不断提升。组织辅导员、班主任参加参加校外和校内培训，在工作中每周、每月进行总结交流，提升团队整体工作能力。针对不同年级学生的特点，形成了大一适应性系列教育、大二学业专业教育、大三求职准备教育、大四毕业生文明离校系列教育活动，辅导员、班主任学生从进校到离校，针对学生群体的不同时期的学业状态和心理特点，开展心理辅导、教育管理工作。2015年辅导员班主任听查课344次，谈心谈话401次，走宿舍289次。

学院倡导以教风促学风，要求任课教师在课堂上敢于管理，严格执行上课每周至少点名一次，节假日前后必须点名的考勤制度，对于迟到、带早餐到课室、上课玩手机等不良现象，及时批评指正并通报给辅导员，并在学院进行教师执行情况的阶段讲评通报。

 

2、目标管理为形成合力打开新思路

在2015年度学院各年级全面认真落实“四位一体”目标管理，全院通过设立年级目标、班级目标、宿舍目标、个人目标，分级重点关注，配合骨干负责制、宿舍公约制度学风和学习成绩均取得较大提升，班级作弊、不及格人次得到有效控制，迟到、旷课、晚归、晚熄灯等总体违纪率保持在较低水平。以2013级为例，2014-2015第二学期期末考试中，挂科率由第三学期的3.89%降至1.74%，效果显著。

 

3、进一步拓展培训渠道，注重实践锻炼

为了弥补辅导员、班主任等学生工作队伍重使用轻培养的弱点，紧紧抓住骨干遴选、业务培训、过程督导、效果评估、总结讲评、述职测评、交流提升等渠道，增进学习交流机会。学院先后选派12人参加省级、校级5项业务培训，分批次、分层次对全体学生骨干进行了轮训，学生教育管理队伍整体素质得到提升。一年来，辅导员队伍中有6人晋级、3人晋升中级职称，获得学校各类优秀奖励6人次，发表学生工作相关论文6篇，参与完成的校级质量工程项目结题1项。2名老师代表学校分别广东省民办高校首届微党课大赛和广东省高校第四届辅导员职业能力大赛，7名班主任被评为学年度优秀班主任，11名学生干部和29名共青团干部获得学校优秀学生干部和优秀共青团干部荣誉。

 

（三）以党建促团建，为优良学风的建设提供强有力的组织保障和思想保障

1、创新党建工作，充分发挥党支部的战斗堡垒和全体共产党员的先锋模范作用

2015年学院党总支接收入党积极分子56名，新发展党员36名，预备党员转正24名。毕业生党组织关系转出53人，教师党组织关系转出2人。共推荐90名优秀学生参加学校第17、18期党课培训班学习。在两期党课学员讲坛中分获一、二等奖。学院党总支和学生第一党支部在2015年均获得学校先进基层党组织荣誉。

2015年学院党总支组织省内参观学习1次，开展校级党建活动5项，组织专题学习15次，在2015年暑期社会实践调研中，由学生第一党支部学生党员组成的调研团队“学生党员形象及社区建设调研”获得了学校暑期社会实践“优秀团队”称号。

针对毕业生外出实习期间的入党积极分子培养、党员教育和党组织活动现状，成功申报2015年度广东省高校党建研究课题1项，并在学生第一党支部建立网络党支部，搭建了基于“互联网+”高年级校外实习实训学生党员教育管理新平台，创建“汽车党员e家”微信公众平台，促进学生党员的学习、管理及交流。截至2015年底，网络党支部成立2个月已开展2次线下党建活动，微信公众平台推送3期线上学习内容，发表相关论文1篇。

学院团委依托新媒体组织开展和宣传重大主题纪念日如“纪念九一八，勿忘国耻，兴我中华”活动线上思想教育活动以及学习两会精神、三月雷锋月、五四青年月、四进四信、社会主义核心价值观“拼字活动”、筑梦中国观影活动等主题教育活动，突出网络对青年进行思想引导教育的趋势。学生第二党支部在2015级新生中，设立“学生带班党员”，协助辅导员、班主任做好所带班级的思想道德建设和优良学风建设，促进学生党员培养的长效机制。为提高人才培养质量为中心的各项工作顺利进行提供了强有力的组织和政治保障。

 

2、强化团组织建设，充分发挥团组织的引领作用

学院团委、学生会在6月份顺利完成团学换届工作，9月份进行基层团支部成立与换届大会，产生33个团支部，配备165名干部，团干部在任职期间无违纪现象。完善组织架构、精简调整部门，组织参与各类培训7次，落实管理制度，增强学生干部素质和能力。在共青团评优工作中，我院团委于5月份被评为2014年度“先进团委”，产生3个校级先进团支部、29名校级优秀共青团干部、177名校级优秀共青团员，同时我院在6月份评选出23名优秀团干部，56名优秀团员先进分子被推荐成为入党积极分子。

2015年度，学院团委举办了团校第二、三期团课培训，共培养150名优秀人才，理论实践紧密结合，较第一期团课新增心理课、团务知识讲座、团日活动策划、团校素拓、参观革命基地国防教育活动等课程项目，加强与推优、党校联系，其中推荐了90名学员入党校学习，发挥党的助手和后备军的作用，努力将团校打造成思想引领、组织建设和培养人才的团组织活动先锋阵地。

 

3、以弘扬汽车文化为核心，彰显汽车十年育人成效

在建校和建院十周年之际，学院成立校友会，密切校友联系，为已毕业的5届3000余名校友搭建校友交流平台，为学院发展提供新一轮动力。积极开拓以汽车文化为核心的学术科技与校园文化活动，学生活动上新台阶。包括营销策划大赛、crazy color汽车彩绘大赛、大学生创意创新方案征集大赛、飞思卡智能车大赛、汽车文化交流形象大使选拨赛、越野体验之旅摄影大赛等组成的汽车文化节系列活动。积极组织和承办学校“Light Run • Like青春”荧光夜跑活动、“筑梦•中国”征文大赛、彩虹人生-奋斗的青春最美丽之职业计划报告会、第七届学院运动会和学院篮球赛与足球赛等文体艺术活动，响应团中央“三走”倡议，丰富了学生的校园文体艺术生活。学院团委为此获表扬组织单位，“Light Run • Like青春”荧光夜跑活动被评为校级优秀活动，彩虹人生-奋斗的青春最美丽之职业计划报告会被评为院级优秀活动。在学校田径运动会和第一届学生组织运动会中，均获男团第一、团体总分第二和道德风尚奖。

 

（四）进一步密切校企合作培养，实现就业质量的有效提升

立足于提高就业质量，加强与企业沟通，强化校企合作，促进人才共育。2015届毕业生专业技能证书考证通过率达到85%。学院有针对性地选择有往届毕业生且合作空间大的企业进行回访，加大力度开发国有大型企业、外商投资企业的资源，集中时间邀请相关企业前来学院招聘人才，为学生提供优势岗位选择机会，进一步提高学生的就业质量。2014年6月份开始为2015届毕业生，学院共邀请33家大型企业到校召开专场招聘会。另一方面，集中安排组织大批量学生到企业参观，了解实际工作环境和工作内容，有选择性地参加面试，提高求职效率；通过与企业沟通，将企业入职培训和岗前培训安排在学校进行，既保证学生安全，又降低了企业的求职成本，使双方受益。

此外，还通过联合申报项目的方式促进校企合作。汽车工程学院现已建成38家就业实习基地。学院与学校工程研究院合作，依托项目带动教学，搭建协同育人实践平台，创新育人模式，开展的2015级车辆工程特色班在优良学风、班风建设及学生能力提升上，已初具成效。

2015届共有毕业生665人，共有635人参加就业，最终就业率为99.85%。考取硕士研究生2人，报名预征入伍1人，已有3人考取了公务员，26名毕业生选择自主创业，其中4人成功创业。学院获得学校就业先进单位一等奖，实现五连冠。

 

三、存在的问题

（一）优良学风工作是一项长期的系统工程，在全员形成合力还需进一步配合，工作执行力和效果需要进一步提升。

（二）学生工作团队能力建设还需要加强，团队活力不足，团队凝聚力需要进一步提升。
